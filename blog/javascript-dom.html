What is the DOM

The DOM (Document Object Model) represents what a website looks like its in code form (HTML, CSS and JavaScript). Its best to think about the DOM as a family tree, with the document as the parent, html, body and text as children cascading down.

The DOM allows JavaScript communicate and interact with HTML and CSS. If our website had a circle that was grey and we had our JavaScript run code that changes the circle's colour to green, JavaScript will find the grey circles in the HTML document using the DOM and with magic change it to green. And by magic I mean JavaScript will manipulate the DOM to change the grey circle to green.
