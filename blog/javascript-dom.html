JavaScript, HTML and CSS

Lets first think about the human body. We can consider the human body as HTML. It’s got a head(er), body and even a foot(er). But as if we leave it as it is, its just bare, and naked. For most people it's embarassing to be naked in front of people. This is where CSS comes in, now CSS are the clothes that a person wears, they can style it however they want.

So where does JavaScript come into this? Let me ask you first, do you think about breathing? Do you actively think about your heart beating? No. Not for me atleast. With CSS and HTML you can navigate pages, change windows sizes and what not without much effort but if we want our body to move, walk, run or jump we have to actively think about it? This is where JavaScript can help! Its pretty much the brains of our webpage. If you’re shopping and you hover over an item to get an image preview, that is JavaScript, or if you add to cart and it is updated in real time, that is JavaScript!

Control Flow and Loops

Which way do you read? Left to right and top to bottom I presume? Control flow is how code is read or processed by a computer. Left to right then top to bottom.

Some times there are certain scenarios that require the code to skip some lines or repeat a big chunk of code. This is where loops come into play. Have you ever had to do something so repetitive like washing lots of dishes? If we had to code washing five different things we can code something like this and it’ll work mostly fine.

See the Pen Sprint 3 Loops1 by JP Guiang (@jp-guiang) on CodePen.

But what if there was a hundred different dishes? Would we want to write that same line of code a hundred times? No right, so what we can do is make a loop instead!

See the Pen Sprint 3 Loops2 by JP Guiang (@jp-guiang) on CodePen.

Instead of saying wash dishes a hundred times now our loop will just keep washing dishes until all the dishes are clean! The great thing about loops is that we can take different user inputs, maybe I want you to wash five dishes, and someone else wants ten dishes cleaned, instead of manually writing those values, our loop can work it out for us instead! I didn’t go into how we can make user inputs in this blog as that is its own beast, for now lets just stick with this.

What are functions?

Remember when I said we that we have to think about jumping, walking or running, Well that is pretty much what JavaScript functions are! We can create code that does specific tasks for us. We can even run code based on certain events like a mouse click, right mouse click, a double click etc etc.

As you can imagine it can be very useful having functions as we can automate certain tasks or do certain things to make the user experience very convenient! An example of a function? Next time you do online shopping and you hit the ‘add to cart button’ there was probably a function called ‘addToCart’ in the code. Imagine if there was no addToCart function! Bye bye online shopping.
