What is the difference between Margin, Border and Padding?

In order to explain the difference between margin, border and padding, let's first imagine that we have a cardboard box.

Inside this cardboard box we can fill it up with whatever we want, toys, stationary, books etc. But if we find that our stuff does not fit in this box we can always get a bigger one, or if the box is too big we can always get a smaller box. Once we finished packing we can close up the box with some tape and maybe even cover it with plastic wrapping.

Now with html and css, we can think of our website as just a bunch of boxes with lots of different things inside of them. They can contain "text", images or links to other pages. If we want to change the size of our boxes, we can manipulate them by changing the "padding" of the box. If we want our box to be made of thicker cardboard we can change it by manipulating "border" Finally if we want to keep our boxes separated from other boxes with tape or plastic, we can use "margin".

See the Pen Untitled by JP Guiang (@jp-guiang) on CodePen.

In summary, a website made from css and html is just a bunch of boxes with stuff inside of them. These boxes all have different sizes, thickness of the box and whether they have tape or plastic on the outside.
